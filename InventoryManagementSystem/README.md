# 📦 Inventory Management System

A simple console-based Inventory Management System built with C# that allows users to manage products, update inventory quantities, and view available products.

## 🚀 Features

* ➕ Add new products
* ✏️ Update product quantity
* 📋 View all products
* 🚪 Exit the application

## 📚 Concepts Demonstrated

### 🗂️ Arrays

Uses a two-dimensional array to store product information:

```csharp
string[,] inventory = new string[50, 3];
```

### 🔄 Loops

* `while` loop for the main menu.
* `for` loop for searching and displaying products.

### 🎯 Conditional Statements

* `if`
* `switch`

### 📥 User Input Handling

* Reading data from the console.
* Converting user input to appropriate data types.

### 🏗️ Data Management

Each product contains:

* Product Name
* Product Quantity
* Product Price
