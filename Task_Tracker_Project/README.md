# ✅ Task Tracker Project

A simple console-based Task Tracker application built with C# that helps users manage daily tasks by adding, viewing, completing, and removing tasks.

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ✅ Mark tasks as completed
* 🗑️ Remove tasks
* 🚪 Exit the application

## 📚 Concepts Demonstrated

### 🗂️ Arrays

Store tasks using a fixed-size array:

```csharp
string[] tasks = new string[100];
```

### 🔄 Loops

Use loops to display all stored tasks and keep the application running.

### 🎯 Methods

Organize functionality into separate methods:

```csharp
AddTasks()
ViewTasks()
TaskCompleted()
RemoveTask()
```

### 📥 User Input Handling

Receive task information and menu selections from the user through the console.

### 🏗️ Task Management Logic

* Create tasks
* Display tasks
* Update task status
* Delete tasks
