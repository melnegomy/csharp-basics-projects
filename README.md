# 🚀 C# Basics Projects

A collection of beginner-friendly C# console applications designed to practice fundamental programming concepts and build a strong foundation in software development.

## 📚 Projects Included

### 🎯 Quiz Game Project

A simple quiz application that asks users a set of questions, validates answers, and calculates a final score.

**Concepts Covered**

* Arrays
* Methods
* Loops
* Conditional Statements
* Exception Handling
* User Input Validation

---

### 📦 Inventory Management System

A console application for managing products in an inventory. Users can add products, update quantities, and view available items.

**Concepts Covered**

* Multidimensional Arrays
* Loops
* Search Algorithms
* Data Storage
* CRUD Fundamentals
* Console Applications

---

### ✅ Task Tracker Project

A task management application that allows users to add, view, complete, and remove tasks.

**Concepts Covered**

* Arrays
* CRUD Operations
* Methods
* Loops
* Conditional Statements
* Console Applications
